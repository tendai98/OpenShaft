# `images/` README

Back to the [root README](../README.md)

This folder contains the repository’s image assets.

## Files currently present

- `1.jpg`
- `2.jpg`
- `3.jpg`
- `app.jpg`

## What this folder is for

These files appear to be the visual reference set for the project:

- photos of the converted RC cars
- app or controller screenshots
- images that can be embedded in the root README or variant-specific READMEs

## Suggested use in the main README

A good future improvement would be to use these images more explicitly:

- one hero image near the top of the root README
- one photo per major hardware variant
- one screenshot for the controller experience
- short captions explaining which build or folder each image belongs to

## Recommended naming improvement

If this repo keeps growing, consider renaming the files to something more descriptive, for example:

- `car-divo-front.jpg`
- `car-rst-side.jpg`
- `controller-app.jpg`
- `build-internals.jpg`

That would make the images easier to reference from documentation.
